# 🔒 단방향 암호

- 암호화 가능
- 복호화 불가능
- 해시를 이용
- SHA 알고리즘이 있음
- SHA1, SHA2, MD5 알고리즘 사용하지 말자 (보안 취약성 발견)
- SHA3(SHA512)을 사용하자

# 🔑 대칭형 암호

- 암호화 키와 복호화 키가 같음
- 대표적으로 DES, AES가 있다

# 🛠 비 대칭형 암호

- 공개키 암호화라고도 한다
- 암호화키와 복호화 키가 다르다
- 암호화키는 공개키라고 한다
- 복호화키는 개인키라고 한다
- RSA 암호화가 대표적이다
